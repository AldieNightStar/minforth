# Variables

## Assign
* To assign variables - use `=name` it will assign new value from stack
```
10 =a
20 =b
30 =c
```

## Read
* To read variables - use `$name` it will read the variable (Not assign)
```
$a
```

## Increment Sample
* We read `a` then adding `1` to it and then assigning back
```
$a 1 + =a
```
